安装过程:

- 对于 `i3/` 和 `picom/` 来讲，只需要将配置文件所在目录复制到 `~/.config/` 即可
- 对于polybar
  - 首先`cp -r polybar_themes ~/.config/polybar/`
  - 然后`cd ~/.config/polybar/`
  - 最后`bash setup_fonts.sh`
